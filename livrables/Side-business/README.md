# Side-business/

Livrables liés aux projets de business : études de marché, business plans, landing pages, contenus, analyses concurrentielles, modèles financiers, etc.

Un sous-dossier par projet de business, avec les livrables datés à l'intérieur.

Exemple :
```
Side-business/
└── formation-management-militaire/
    ├── 2026-05-31_etude-marche.md
    └── 2026-06-10_landing-page/
```
